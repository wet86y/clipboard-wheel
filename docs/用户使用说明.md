# 超级中键 使用说明

---

## 这是什么？

**超级中键**是一个 Windows 剪贴板增强工具。它的核心操作方式是：

> **按住鼠标中键（滚轮）→ 出现轮盘 → 手指移到你想粘贴的方向 → 松开中键**

不需要切窗口、不需要 `Ctrl+V` 翻历史、不需要眼睛离开屏幕。特别适合大量复制粘贴的场景，比如写文档、填表格、聊微信、写代码。

---

## 怎么安装 / 运行？

- **绿色免安装**，一个 `超级中键.exe` 文件即可运行。
- **不需要管理员权限**。
- 双击 exe 后，任务栏右下角托盘会出现一个蓝色或灰色的圆形图标，程序就开始工作了。
- 配置文件会自动保存到 `C:\Users\<你的用户名>\AppData\Roaming\超级中键\settings.json`，不用担心丢失。
- 除设置文件外，轮盘历史、窗口接管会话和临时预览均只保留在当前进程内；退出后会释放，不会留下额外运行数据。正式包不会写运行日志。

## 核心功能

### 1. 基础剪贴板轮盘

| 操作 | 效果 |
|------|------|
| **按住中键** | 在鼠标位置呼出一个圆形或矩形轮盘 |
| **移动鼠标到某个扇区** | 该扇区高亮放大，预览剪贴板内容（文字 / 图片 / 表格） |
| **松开中键** | 把该扇区对应的剪贴板内容粘贴到当前光标位置 |
| **按 Esc** | 取消当前轮盘并立即恢复鼠标，不会关闭后续中键监听 |

- 轮盘默认显示最近 6 条（可在设置里改成 4 或 8 条）。
- 最后一个扇区是 **"复制"** 按钮（深绿色），相当于帮你按 `Ctrl+C`，选中即复制。
- 轮盘形态支持**圆形 4/6/8 扇区**和**矩形 4/8 扇区**，在设置里切换。
- **没有历史也能打开空轮盘**——刚装完就能用。
- **单次轮盘最多保持 15 秒**。到时只取消当前轮盘并释放鼠标，之后仍可再次按中键使用。
- 普通权限运行时如果切换到管理员窗口或 UAC 安全桌面，本次轮盘会自动取消；回到普通窗口后自动恢复。

### 2. 突破轮盘（圆形模式专属）

圆形模式下，把鼠标再往外拖一下，轮盘外面会再弹出一圈**动作按钮**：

```
方向盘方向（上、下、左、右）
每组 3 个槽位 = 共 12 个可配置槽位
```

突破轮盘**不放剪贴板**，而是放你常用的工具：

- **快捷键模式**：比如录一个 `Ctrl+Shift+S`（Windows 截图），选中后自动发送到当前窗口。
- **快捷方式模式**：直接拖入程序、文件、文件夹或已有 `.lnk`，选中后快速打开。再触发一次可以最小化或关闭。

**空槽不会干扰你**：如果鼠标移到了没绑定任何动作的突破槽位，会自动回到内圈普通扇区。

### 3. 临时锁定扇区

有时某个剪贴板内容你反复要用，但又怕新复制的内容把它顶掉。

- 按住中键打开轮盘
- 把鼠标移到你想锁定的历史扇区上
- **右键点一下** → 选中扇区变蓝灰色 = 已锁定
- 再右键同一扇区 → 解除锁定

锁定只对当前程序运行有效，关掉重开就不锁了。

### 4. 图片复制预览

在设置里打开"图片捕获"后，截图、复制的图片会自动进入轮盘历史。

- 轮盘里会显示**缩略预览图**，看一眼就知道是哪张图。
- 选中后粘贴的是**原图**，不会降清晰度。
- 图片超边会自动裁剪，不会溢出到隔壁扇区。

### 5. 多格式粘贴

程序支持多种剪贴板格式：

| 格式 | 说明 |
|------|------|
| 纯文本 | 字面粘贴 |
| HTML / 富文本 | 保留颜色、字体 |
| 表格（含 Excel / WPS） | 保留单元格格式 |
| CSV / TSV | 表格数据识别 |
| RTF | 兼容格式 |
| 图片 | 可选开启，见设置 |

如果经常把 Excel/WPS 的一行单元格粘贴到文本框或双击后的单元格编辑区，可以在常规设置中打开**表格文本清洗**。它只清洗表格内容的纯文本粘贴，删除横向空白和中英文引号并保留换行；选中表格区域进行正常格式化粘贴时，单元格结构和格式不变。

## 设置说明

右键点击托盘图标 → **设置**，可以看到以下页面：

### 常规设置
- 轮盘形状（圆形 / 矩形）
- 扇区数量（4 / 6 / 8）
- 轮盘大小
- 中心死区（核心圈大小）
- 透明度
- 图片捕获开关
- 表格文本清洗（默认关闭；仅清洗表格内容的纯文本粘贴，保留换行）
- 开机自启动（默认关闭；开启后登录 Windows 会自动运行当前用户的超级中键）

### 轮盘设置（突破轮盘）

这里有一个**缩小版轮盘预览**，你可以直接看到自己的 12 个外圈槽位。

1. **点击外圈槽位**选中它
2. 在下方的配置栏设置：

   - **模式下拉**：选择"快捷键"还是"快捷方式"
   - **名称**：给按钮起个名（快捷键模式必须填；快捷方式可不填，不填就显示程序名）

   如果选了**快捷键模式**：
   - 点"录制"→ 按下你想绑的键盘组合键（如 `Ctrl+Shift+S`）→ 点"结束录制"
   - 右边显示区会显示录好的快捷键
   - 之后点"清除"可以删掉

   如果选了**快捷方式模式**：
   - 把程序、普通文件、文件夹或 Windows 快捷方式 `.lnk` 直接拖到设置栏
   - 普通路径会自动转换为由超级中键统一保管的 `.lnk`；已有 `.lnk` 仍按原路径使用，不会被复制或修改
   - 管理员模式运行时，点击空槽拖入框会打开一个普通权限的拖放窗口；把上述任一项目拖入该窗口即可安全交回主程序
   - 会自动显示程序图标和名称
   - 点"清除"可以解除绑定
   - 如果快捷方式指向浏览器，会显示“浏览器启动网址”输入框；填入网址后，每个槽位可独立打开自己的网址，同时沿用浏览器原有 Profile、登录态和 Cookie

3. **改完多个槽位后，记得点底部的"保存"**。关闭窗口未保存的修改会丢弃。

4. 页面顶部有**突破轮盘总开关**和**边界缓冲区**。

## 托盘图标说明

| 图标颜色 | 状态 |
|----------|------|
| **蓝色圆** | 中键轮盘功能已开启 |
| **灰色圆** | 中键轮盘功能已关闭（暂停捕获） |

- **左键单击图标**：快速开关中键捕获
- **右键单击图标**：弹出菜单（开关、设置、清空历史、退出）
- **打开设置**：只通过右键菜单里的"设置"进入；重复点"设置"只会激活已有窗口，不会打开多个。

## 小技巧 & 注意

### 好用的
- **清空历史**：托盘右键菜单里有，用完想"归零"就清一下
- **中键轮盘关了不影响剪贴板监听**：不想用轮盘时左键点一下托盘图标就行，不用退出
- **轮盘在 PPT 里用不会翻页**：程序专门拦截了滚轮误触，不会跳到下一页
- **Excel 表格粘贴保留格式**：程序会自动识别表格结构，不会变成一坨纯文本
- **无历史也能开轮盘**：刚装好、没复制过东西，中键也能唤出空轮盘，不报错
- **应用二次触发**：只会控制由该槽位启动并成功识别的窗口，不会扫描或关闭你手动打开的同名应用窗口

### 注意
- **中键就是滚轮按下**——不是滚动，是按下去。不习惯的话多试几下，很快手的。
- **关闭程序不会丢失历史**：当前版本剪贴板历史不存盘，重启后清空，但 Win+V 历史会从系统导入补底。
- **不要在同一个文件夹里同时开两个**：程序是单实例的，第二次双击不会重复启动。
- **发给朋友用的话**：把整个打包目录（如 `超级中键-win-x64`）压缩发过去就行，对方解压后双击 `超级中键.exe` 即可。

## 系统要求

- **操作系统**：Windows 10 / Windows 11（64 位）
- **运行环境**：不需要装 .NET，exe 自带所有依赖
- **权限**：普通用户权限即可，不需要管理员；如目标程序以管理员身份运行，可在设置页打开“管理员模式运行”，点击“保存”后重启生效。是否出现 UAC 确认框由 Windows 的 UAC、管理员批准模式和程序兼容性设置决定；没有弹窗不等于程序没有提升。部分关闭 UAC 或始终使用完整管理员令牌的系统无法切回普通权限，此时应以任务管理器显示的实际“提升”状态为准。

Windows 7 / 8 理论上也能运行，但有一些图片格式兼容性问题，可能需要额外注意截图工具的格式选择。
