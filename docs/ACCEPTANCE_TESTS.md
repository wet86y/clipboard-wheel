# 超级中键验收标准

## A. 基础运行

- [ ] 双击 exe 可启动。
- [ ] 托盘出现图标。
- [ ] 首次启动打开设置窗口。
- [ ] 再次启动默认不强制打开设置窗口。
- [ ] 托盘菜单可打开设置窗口；重复点击“设置”只激活已有窗口，不创建多个窗口。
- [ ] 双击托盘图标不会打开设置窗口；设置只能从右键菜单进入。
- [ ] 托盘菜单可退出程序。

## B. 剪切板监听

- [ ] 复制纯文本后进入历史。
- [ ] 最近复制内容排在前面。
- [ ] 重复内容去重并提前。
- [ ] 超过最大历史数量后移除最旧条目。
- [ ] 图片捕获关闭时，复制图片不进轮盘。
- [ ] 图片捕获开启时，复制图片进入轮盘并显示缩略图。
- [ ] Excel 内表格复制不误存为图片条目。
- [ ] 不同来源图片（画图、浏览器、聊天软件、截图工具）右键复制后均能进入轮盘。

## C. Win+V 历史导入

- [ ] 系统开启剪切板历史时，启动后尝试导入。
- [ ] 系统未开启时，程序不崩溃。
- [ ] 导入失败时，运行期剪切板监听仍可用。
- [ ] 启动导入进行中复制新内容，新内容排在最前；后续导入项只补到底部空位，不顶掉当前复制内容。

> 备注：Wheel paste 进不进 Win+V 是 **best-effort**。原生 `IDataObject` 会公开
> `CanIncludeInClipboardHistory=false`，但目标应用仍可能重新写入剪贴板历史；这不影响本地自捕获抑制。

## D. 中键轮盘

- [ ] 按住中键显示轮盘。
- [ ] 首次启动且没有读取到任何历史时，按住中键仍显示空轮盘。
- [ ] 鼠标移动可高亮对应扇区。
- [ ] 中心死区内松开取消。
- [ ] 扇区外松开粘贴选中内容。
- [ ] 托盘暂停后中键不弹轮盘。
- [ ] 恢复后中键轮盘正常。
- [ ] 轮盘显示期间按 Esc 只取消当前轮盘并立即释放鼠标；随后仍可再次按中键唤起。
- [ ] 单次轮盘会话达到 15 秒时自动取消并释放中键、左键和右键状态；不会永久关闭中键监听。
- [ ] 轮盘显示期间切换到更高权限窗口或 UAC 安全桌面时立即取消且左右键可用；回到普通窗口后可再次按中键唤起。
- [ ] 在 PowerPoint 中按住中键呼出轮盘时，轻微滚动滚轮不会切换到下一页或让当前编辑焦点丢失。
- [ ] 选中普通历史扇区后右键，扇区变为蓝灰色锁定；松开中键只关闭轮盘，不粘贴。
- [ ] 锁定第一个历史扇区后复制新内容，新内容进入第二个可用历史扇区。
- [ ] 再次选中已锁定扇区并右键，锁定解除，后续新内容可回到该槽位。
- [ ] Quick Copy 扇区右键不锁定、不变色，松开中键仍执行 Ctrl+C。
- [ ] 锁定靠后扇区后切换到更少扇区/切换矩形或圆形，锁定项会压回当前可见历史槽位，Quick Copy 仍占最后复制扇区。
- [ ] 锁定状态只在当前程序运行期有效，退出并重新启动后不保留锁定。

## E. 自动粘贴

- [ ] 记事本可粘贴。
- [ ] 浏览器输入框可粘贴。
- [ ] WPS / Word 可粘贴。
- [ ] Excel 单元格输入区可粘贴。
- [ ] 目标程序管理员运行时，普通权限无法粘贴不视为 bug；管理员重启后应可验证。

> 备注：`restoreClipboardAfterPaste` 固定为 **false**，避免与 Excel/WPS 慢读产生竞态（“开始好用过一会退回”）。详见 `docs/DESIGN_NOTES.md` §7。

## F. 突破轮盘

- [ ] 突破轮盘总开关关闭时，任何圆形/矩形模式都不会唤出扩展扇区。
- [ ] 圆形 4 / 6 / 8 模式下，鼠标突破原轮盘上方外缘时显示上方 3 个扩展扇区。
- [ ] 圆形 4 / 6 / 8 模式下，鼠标突破原轮盘下方外缘时显示下方 3 个扩展扇区。
- [ ] 圆形 4 / 6 / 8 模式下，鼠标突破原轮盘左方外缘时显示左方 3 个扩展扇区。
- [ ] 圆形 4 / 6 / 8 模式下，鼠标突破原轮盘右方外缘时显示右方 3 个扩展扇区。
- [ ] 矩形 4 / 8 模式下，鼠标突破轮盘外缘也不会显示扩展扇区。
- [ ] 鼠标回到原轮盘覆盖范围内时，扩展扇区收起。
- [ ] 边界缓冲区能减少扩展扇区在边缘反复闪烁。
- [ ] 扩展扇区从原轮盘外缘弹出，动画速度与现有轮盘唤起速度接近。
- [ ] 从一个已展开方向切换到外围另一方向时，新方向的扇区和文字同样有淡入、缩放入场动画。
- [ ] 外圈应用/快捷键名称不再按固定字符数截断；只有实际被遮罩超过两个完整字符时才连续循环滚动，短名称或轻微裁切保持静止。
- [ ] 未绑定扩展槽显示为淡化空槽，不显示“未设置”文字。
- [ ] 鼠标落在未绑定扩展槽方向时，该空槽不会高亮、放大或执行动作，并回落到内圈普通扇区选择。
- [ ] 松开中键时如果鼠标在原轮盘扇区，执行原剪贴板/Quick Copy 逻辑。
- [ ] 设置页显示完整缩小版轮盘 UI，包含内圈原始轮盘和外圈 12 个扩展动作槽。
- [ ] 缩小版轮盘随当前轮盘半径、中心死区、圆形扇区数变化；矩形模式下仍可配置但不触发。
- [ ] 设置页只能选择外圈扩展槽，选中槽用默认蓝色覆盖。
- [ ] 快捷键模式通过“录制/清除”管理组合键，录制时截断键盘输出，键位显示区域不可直接编辑。
- [ ] 快捷键模式录制组合键后能向当前前台窗口发送该组合键；名称为空时轮盘可显示快捷键文本。
- [ ] 应用模式可拖入 `.lnk`，拖入后显示快捷方式图标和名称，绑定后必须先清除才能再次导入。
- [ ] 管理员模式运行时，点击应用模式空槽的拖入框会打开普通权限辅助窗口；从普通资源管理器拖入 `.lnk` 后主设置页完成导入，非 `.lnk` 或已绑定槽位不得导入。
- [ ] 快捷方式名称为空时显示快捷方式名称。
- [ ] 快捷方式模式轮盘显示上方图标、下方名称。
- [ ] 快捷方式第一次触发打开/唤起目标，第二次触发按设置关闭或最小化。
- [ ] 同一浏览器可绑定多个不同网址；每个槽位首次触发打开对应网址，且保留本机浏览器 Profile、登录态和 Cookie。
- [ ] 文档、PDF、Excel/Word/PowerPoint 的二次触发只控制该槽位已识别的窗口，不影响用户手动打开的同名窗口。
- [ ] WPS 标签页复用时，二次触发不会新增临时小窗口，也不会关闭整个共享 WPS 宿主。
- [ ] 文件夹窗口手动关闭后，再次触发对应槽位会重新打开该文件夹。
- [ ] Everything 等“关闭后缩到托盘”的程序，二次触发会恢复并置前；恢复后再次触发才按配置关闭或最小化。
- [ ] 单实例或后台常驻应用在没有可验证目标窗口时，短等待期后再次触发可重新按快捷方式唤起，不会被常驻进程永久阻塞。
- [ ] 设置页外圈 12 个扩展槽位与实际上/下/左/右方向映射一致。
- [ ] 设置页所有开关控件为红绿拨轮/滑动开关，不使用复选框。

## G. Excel / WPS 表格

- [ ] 单单元格复制后，在普通文本框粘贴为纯文本。
- [ ] 多单元格复制后，PlainText 至少为 TSV 结构。
- [ ] 多单元格复制后，在支持富文本的目标中尽量保留格式。
- [ ] Smart 模式自动选择：单格文本走纯文本，表格/富文本保留可用格式。

## H. 设置

- [ ] 可调整扇区数量。
- [ ] 可调整轮盘半径。
- [ ] 可调整死区范围。
- [ ] 可调整透明度。
- [ ] 设置窗口保持单页，可滚动，包含轮盘视觉/布局设置和图片捕获开关。
- [ ] 快捷复制默认开启，仍可在设置中关闭。
- [ ] 图片捕获默认关闭；打开后复制图片会进入轮盘并显示缩略图。
- [ ] 图片粘贴到 Excel / Word / 聊天窗口均可正常显示。
- [ ] 设置保存到 `%AppData%\超级中键\settings.json`。
- [ ] 开机自启动默认关闭；保存开启后仅在当前用户 `Run` 注册表项创建超级中键启动命令，保存关闭后该命令被删除。
- [ ] 配置损坏时自动备份并回退默认配置。
- [ ] 重复启动程序时不会创建第二个托盘图标、第二个鼠标钩子或第二个剪贴板监听器。第二个进程直接静默退出。

## I. 发布

- [ ] 可发布 single-file exe。
- [ ] 不需要安装包。
- [ ] 不需要管理员权限也能正常运行基础功能。
- [ ] 设置页的“管理员模式运行”开关只在点击“保存”后生效，并按需重启程序。

## J. Win+V 污染控制

> 后续场景验证，依靠场景 A 跳过 SetDataObject 优化（“复制 X → wheel 选 X 再粘”不调 SetDataObject、Win+V 不增条目）以及固定关闭的 `addPasteToClipboardHistory`（wheel 粘贴不主动上报 Win+V）。详见 `docs/DESIGN_NOTES.md` §6 + §9。

- [ ] 复制 X 后 wheel 选 X → Win+V 中不出现新的 wheel paste 条目。
- [ ] 复制 X 后 wheel 选 Y（不同内容）→ Win+V 中出现 Y 的条目。
- [ ] 交替复制 A/B/C/D ...（中间不粘贴）→ Win+V 条目数 ≤复制次数，以 Win+V 自身的丢包为准。
