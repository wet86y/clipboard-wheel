# 超级中键 2.1.1

本版本增加表格内容的可选纯文本清洗，并支持将程序、普通文件和文件夹直接拖入突破轮盘。

- 基础设置新增默认关闭的“表格文本清洗”开关；仅清洗已识别表格的纯文本载荷，移除横向空白和中英文单双引号，同时完整保留换行。
- Excel/WPS 表格区域粘贴继续保留 HTML、RTF、CSV 等原格式；进入单元格编辑状态或粘贴到文本框时使用清洗后的纯文本。
- 突破轮盘应用槽支持直接拖入单个程序、普通文件或文件夹，并自动生成受管 `.lnk`；原有外部 `.lnk` 仍直接引用。
- 受管快捷方式采用事务式保存与严格目录边界清理，取消编辑、保存失败、重新绑定或清除槽位时不会删除源文件或用户外部快捷方式。
- 正式 Release 继续保持单 EXE，不包含诊断日志、PDB 或故障注入。
